VFP code example
DIMENSION myArray(2,3)    && create an array of 2 rows and 3 columns
? ALEN(myArray)           && returns 6 (2*3 elements)
? ALEN(myArray,1)         && returns 2 (rows)
? ALEN(myArray,2)         && returns 3 (columns)
